Ball Machine

The objective of this project was to design and build a device that would infinitely cycle and sort steel ball bearings. All the machines were tested in a competetion to see which cycled closed to 40 balls/minute the quietest. My group certainly did not have the prettiest ball machine but it did work and we were proud of that!

Goals

  • Develope Design Concepts
  • Decision Matrix to select best design
  • Fully model design in Solidworks
  • Build a working prototype and make improvments as necessary
